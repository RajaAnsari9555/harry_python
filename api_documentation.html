📝 Registration
POST /auth/register Auth

Naya user register karo. Success hone pe OTP user ke email pe bheja jata hai.

Request Body (JSON)
FieldTypeRequiredDescription
namestringRequiredUser ka poora naam
emailstring (email)RequiredValid email address
passwordstringRequiredAccount ka password
{ "name": "Ali Khan", "email": "ali@example.com", "password": "mypassword123" }
Responses
200 OK {"message": "Registration successful. OTP sent to your email."}
400 {"detail": "Email already registered"}
POST /auth/register/verify-otp Auth

Registration ke baad email pe aaya OTP verify karo. Verify hone pe account activate ho jata hai.

Request Body (JSON)
FieldTypeRequiredDescription
emailstring (email)RequiredRegistered email address
otpstringRequired6-digit OTP jo email pe aaya
{ "email": "ali@example.com", "otp": "483920" }
Responses
200 OK {"message": "Account verified successfully"}
404 {"detail": "User not found"}
400 {"detail": "Account already verified"}
400 {"detail": "Invalid or expired OTP"}
🔵 Login (OTP-based)
POST /auth/login Auth

Login ke liye OTP request karo. Sirf email chahiye — OTP email pe bheja jayega.

Request Body (JSON)
FieldTypeRequiredDescription
emailstring (email)RequiredRegistered aur verified email
{ "email": "ali@example.com" }
Responses
200 OK {"message": "OTP sent to your email for login"}
404 {"detail": "User not found"}
403 {"detail": "Account not verified"}
POST /auth/login/verify-otp Auth

Login OTP verify karo. Valid hone pe JWT Access Token return hota hai — ise aage ke requests mein use karo.

Request Body (JSON)
FieldTypeRequiredDescription
emailstring (email)RequiredRegistered email address
otpstringRequired6-digit OTP jo email pe aaya
{ "email": "ali@example.com", "otp": "739201" }
Responses
200 OK {"access_token": "eyJhbGci....", "token_type": "bearer"}
404 {"detail": "User not found"}
400 {"detail": "Invalid or expired OTP"}

Request Body (JSON)
FieldTypeRequiredDescription

POST /auth/forgot-password/reset Auth

OTP verify karke naya password set karo.

Request Body (JSON)
FieldTypeRequiredDescription
emailstring (email)RequiredAccount ka email address
otpstringRequired6-digit OTP jo email pe aaya
new_passwordstringRequiredNaya password jo set karna hai

Responses
200 OK {"message": "Password reset successfully"}
404 {"detail": "User not found"}
400 {"detail": "Invalid or expired OTP"}
📌 General Notes
  • Saare endpoints Content-Type: application/json expect karte hain
  • OTP sirf 10 minutes valid hota hai — expire hone pe naya mangwana hoga
  • JWT token 30 minutes mein expire hota hai
  • Login ke baad mila access_token protected routes mein Authorization: Bearer <token> header mein bhejo
  • Interactive docs: http://localhost:8000/docs (Swagger UI)
