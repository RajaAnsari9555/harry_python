📌 Async / Await — Ek saath kai kaam karna

Yeh thoda advanced hai — jab koi kaam time leta hai (jaise database se data lana, email bhejna), toh async use karte hain taaki baaki kaam ruka na rahe.

import asyncio async def email_bhejo(): print("Email bhej raha hoon...") await asyncio.sleep(2) # 2 second wait (simulate karna) print("Email bhej diya!") asyncio.run(email_bhejo())
💡 Hamare project mein saare controller functions async def hain kyunki woh MongoDB aur email ke saath kaam karte hain.

5. Project mein Use Ki Gayi Libraries — Ek Ek Karke

🚀 FastAPI

Kya hai: Python ka ek web framework — matlab iska use karke tum API (backend server) bana sakte ho jo internet pe requests receive kare aur response de.

from fastapi import FastAPI app = FastAPI() @app.get("/") # jab koi "/" pe aaye async def root(): return {"message": "Hello World"} # yeh response milega
💡 @app.get("/") ek decorator hai — matlab yeh function tab chalega jab koi us URL pe aayega.

⚡ Uvicorn

Kya hai: Yeh ek server hai jo FastAPI app ko run karta hai. Jaise ek dukaan ka darwaza — bina iske koi andar nahi aa sakta.

uvicorn app.main:app --reload # app.main = app/main.py file # app = us file mein FastAPI object ka naam # --reload = code change hone pe auto restart

🗄️ Motor + PyMongo

Kya hai: Motor ek async MongoDB driver hai — Python se MongoDB database se baat karne ka tarika. PyMongo iska base hai.

from motor.motor_asyncio import AsyncIOMotorClient client = AsyncIOMotorClient("mongodb://localhost:27017") db = client["auth_db"] users = db["users"] # Data insert karna await users.insert_one({"naam": "Ali", "email": "ali@gmail.com"}) # Data dhundna user = await users.find_one({"email": "ali@gmail.com"}) # Data update karna await users.update_one({"email": "ali@gmail.com"}, {"$set": {"umar": 11}})

🔒 python-dotenv

Kya hai: .env file se secret values (passwords, keys) load karne ke liye. Taaki code mein directly nahi likhna pade.

from dotenv import load_dotenv import os load_dotenv() # .env file padho secret = os.getenv("SECRET_KEY") # value lo print(secret) # mysecret

✅ Pydantic

Kya hai: Data validate karne ke liye. Jaise agar koi email field mein "abc" bheje (jo valid email nahi), toh Pydantic automatically error de deta hai.

from pydantic import BaseModel, EmailStr class RegisterRequest(BaseModel): naam: str email: EmailStr # valid email hona chahiye password: str # Agar koi galat data bheje toh FastAPI khud error return karega # {"detail": "value is not a valid email address"}

🔑 Passlib + bcrypt

Kya hai: Password ko hash karne ke liye — matlab plain password ko ek aisi string mein badalna jo wapas nahi padhi ja sake. Database mein kabhi plain password nahi rakhte.

from passlib.context import CryptContext pwd = CryptContext(schemes=["bcrypt"], deprecated="auto") # Hash banana hashed = pwd.hash("mera_password123") print(hashed) # $2b$12$abc...xyz (yeh store hoga database mein) # Verify karna result = pwd.verify("mera_password123", hashed) print(result) # True
⚠️ Hash ek taraf ka kaam hai — hash se original password nahi nikala ja sakta. Isliye secure hai!

🎫 python-jose (JWT)

Kya hai: JWT (JSON Web Token) banane ke liye. Login ke baad user ko ek token milta hai — jab bhi woh koi protected API call kare, yeh token bhejta hai taaki server jaane ki woh logged in hai.

from jose import jwt from datetime import datetime, timedelta SECRET = "mysecret" # Token banana payload = {"sub": "ali@gmail.com", "exp": datetime.utcnow() + timedelta(minutes=30)} token = jwt.encode(payload, SECRET, algorithm="HS256") print(token) # eyJhbGci... (yeh token user ko milta hai) # Token decode karna data = jwt.decode(token, SECRET, algorithms=["HS256"]) print(data["sub"]) # ali@gmail.com
💡 JWT token ek sealed envelope jaisa hai — andar data hai, aur sirf sahi key se khul sakta hai.

📧 fastapi-mail

Kya hai: FastAPI se email bhejne ke liye. Hamare project mein OTP email pe bheja jata hai isi library se.

from fastapi_mail import FastMail, MessageSchema, ConnectionConfig conf = ConnectionConfig( MAIL_USERNAME="example@gmail.com", MAIL_PASSWORD="app-password", MAIL_FROM="example@gmail.com", MAIL_PORT=587, MAIL_SERVER="smtp.gmail.com", MAIL_STARTTLS=True, MAIL_SSL_TLS=False, USE_CREDENTIALS=True, ) async def email_bhejo(to_email, otp): message = MessageSchema( subject="Tumhara OTP", recipients=[to_email], body=f"Tumhara OTP hai: {otp}", subtype="plain", ) fm = FastMail(conf) await fm.send_message(message)
